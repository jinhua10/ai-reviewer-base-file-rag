## 🔄 API 对接

### 请求参数映射

| 前端状态 | 后端参数 | 类型 | 默认值 |
|---------|---------|------|--------|
| currentPage | page | int | 1 |
| pageSize | pageSize | int | 20 |
| sortBy | sortBy | string | "date" |
| sortOrder | sortOrder | string | "desc" |
| filterText | search | string | "" |

### 响应数据映射

| 后端字段 | 前端使用 | 说明 |
|---------|---------|------|
| total | 显示统计 | 总文档数 |
| documents | 渲染列表 | 文档数组 |
| page | 页码显示 | 当前页 |
| pageSize | - | 确认每页数 |
| totalPages | 计算分页 | 总页数 |

1. **无需额外操作**

   - 刷新页面即可生效

2. **验证**
   ```bash

   ```

### 后端部署

1. **重新编译**
   ```bash
   mvn clean package
   ```

2. **重启服务**
   ```bash
   java -jar target/ai-reviewer-*.jar
   ```
