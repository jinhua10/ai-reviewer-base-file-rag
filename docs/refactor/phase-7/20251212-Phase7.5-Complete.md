## 📚 使用示例

### 暗色主题切换
```javascript
import { useApp } from '@hooks'

function ThemeToggle() {
  const { theme, toggleTheme } = useApp()
  
  useEffect(() => {
    document.documentElement.setAttribute('data-theme', theme)
  }, [theme])
  
  return (
    <Button onClick={toggleTheme}>
      {theme === 'light' ? '🌙 暗色' : '☀️ 亮色'}
    </Button>
  )
}
```

### 使用动画
```jsx
// 使用动画类
<div className="animate-fadeInUp">内容</div>

// 使用悬停效果
<Card className="hover-lift">卡片</Card>

// 使用过渡效果
<div className="transition-all">内容</div>
```

### 使用响应式
```jsx
// 响应式显示/隐藏
<div className="d-none d-md-block">桌面端显示</div>
<div className="d-block d-md-none">移动端显示</div>

// 响应式网格
<div className="row">
  <div className="col-12 col-md-6 col-lg-4">列</div>
</div>

// 响应式间距
<div className="p-responsive">内容</div>
```

### 使用工具类
```jsx
// 间距
<div className="mt-lg mb-md p-sm">内容</div>

// Flex布局
<div className="d-flex justify-center align-center">
  居中内容
</div>

// 文本样式
<p className="text-center text-primary font-bold">文本</p>

// 背景和阴影
<div className="bg-secondary shadow-md rounded-lg p-md">
  卡片
</div>
```
