## 🚀 技术亮点

### 1. 选择 Vite 而非 Webpack ✨
```yaml
优势:
  ✅ 开发启动速度极快（基于 ESM）
  ✅ 配置简单直观（~70 行代码）
  ✅ 热更新速度快
  ✅ 构建体积小
  ✅ 对 React 支持完善

对比:
  Vite 启动: ~1 秒
  Webpack 启动: ~10 秒
```

### 2. CSS 变量系统 ✨
```yaml
功能:
  ✅ 主题色统一管理
  ✅ 间距、圆角、阴影标准化
  ✅ 易于扩展暗色主题
  ✅ 提升样式复用性

定义:
  --primary-color
  --spacing-md
  --radius-lg
  --transition-base
  ... 共 20+ 变量
```

### 3. 路径别名配置 ✨
```yaml
别名:
  @ → src/
  @components → src/components/
  @api → src/api/
  @hooks → src/hooks/
  @contexts → src/contexts/
  @utils → src/utils/
  @styles → src/styles/
  @assets → src/assets/

优势:
  ✅ 导入路径更简洁
  ✅ 避免相对路径地狱（../../）
  ✅ IDE 自动补全支持
```

### 4. API 代理配置 ✨
```yaml
配置:
  开发环境: http://localhost:3000
  API 代理: /api → http://localhost:8080

优势:
  ✅ 解决跨域问题
  ✅ 前后端分离开发
  ✅ 无需修改 API 调用代码
```

### 快速开始

#### 1. 安装依赖（已完成）✅
```bash
cd UI
npm install
```

#### 2. 启动开发服务器
```bash
npm run dev
```
- 访问：http://localhost:3000
- 自动热更新
- API 自动代理到 http://localhost:8080

#### 3. 生产构建
```bash
npm run build
```
- 输出到：`../src/main/resources/static/`
- 自动优化压缩
- 生成 sourcemap

```bash
npm run lint
```
