## 💡 开发提醒

### ✅ 必须遵守
1. **JSX格式**: 所有组件使用 JSX
2. **样式分离**: 禁止内联样式
3. **BEM命名**: CSS类名规范
4. **国际化**: 使用 t() 函数
5. **React Hooks**: 优先使用 Hooks

### ❌ 禁止事项
1. 内联样式 `style={{}}`
2. 硬编码文本
3. `var` 声明
4. 直接操作 DOM
5. 索引作为 key
