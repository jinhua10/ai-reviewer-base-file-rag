### 技术要点
```yaml
流式输出:
  - 使用 EventSource（SSE）接收流式数据
  - 使用 useState + useEffect 逐字渲染
  - 实现打字机效果

Markdown 渲染:
  - 使用 marked 库解析 Markdown
  - 使用 highlight.js 代码高亮
  - 自定义渲染器（支持表格、图片、链接）

样式规范:
  - ⚠️ 禁止内联样式
  - 所有样式提取到 qa/*.css
  - 使用 BEM 命名法
  - 使用 CSS 变量（主题色、间距）
```
