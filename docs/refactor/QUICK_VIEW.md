### 编码规范速记

**后端规范**:
```java
// 1. 注释格式
/** 方法说明 (Method Description) */

// 2. Lombok 优先
@Data  // 优先使用
public class MyClass {
    private String field;
}

// 3. 日志国际化
log.info(I18n.get("module.key"));

// 4. 字符串提取到 YAML
src/main/resources/i18n/module.yml
```

**前端规范**:
```jsx
// 1. JSX 优先（不用纯 JS 创建 DOM）
function MyComponent({ title }) {
    return <div className="my-component">{title}</div>;
}

// 2. 样式分离（禁止内联样式）
// ❌ <div style={{color: 'red'}}>Bad</div>
// ✅ <div className="error-text">Good</div>

// 3. 文件命名
MyComponent.jsx  // PascalCase + .jsx

// 4. CSS 类名
.my-component__header--active  // BEM 命名
```
