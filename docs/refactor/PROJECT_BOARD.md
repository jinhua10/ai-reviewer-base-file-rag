## 💡 今日提示 (Daily Tip)

```
💡 编码规范提醒:
   优先使用 @Data 注解，避免手写 getter/setter
   
示例:
@Data
public class User {
    private String userId;
    private String userName;
}

而不是手写几十行的 getter/setter 方法！
```
