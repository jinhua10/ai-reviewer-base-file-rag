## 🎬 AI 视频生成步骤

### 方案 A：使用 HeyGen/Synthesia 生成数字人视频

#### 步骤 1：准备脚本
```
1. 使用上面的脚本模板
2. 或者用 ChatGPT 生成定制脚本：

提示词：
"请为我写一个 60 秒的产品介绍视频脚本。
产品是'智能知识库助手'，一个企业知识管理 AI 系统。
核心卖点：智能问答、越用越聪明、安全共享、AI 服务扩展。
目标受众：企业管理者和员工。
风格：专业但不枯燥，有感染力。"
```

#### 步骤 2：登录 HeyGen
```
1. 访问 https://www.heygen.com/
2. 注册/登录账号
3. 点击 "Create Video"
```

#### 步骤 3：选择数字人
```
1. 选择 "Avatar" 类型
2. 推荐选择：
   - 亚洲面孔（如 Angela、Michael）
   - 商务装扮
   - 自然表情
3. 或上传自己的照片生成专属数字人
```

#### 步骤 4：输入脚本
```
1. 将脚本粘贴到文本框
2. 选择语言（中文）
3. 选择声音（推荐专业、沉稳的声音）
4. 调整语速（建议 1.0-1.1x）
```

#### 步骤 5：添加素材
```
1. 添加背景（办公室场景）
2. 添加产品截图/录屏
3. 添加 Logo 水印
4. 添加字幕
```

#### 步骤 6：生成视频
```
1. 预览效果
2. 调整细节
3. 点击 "Generate"
4. 等待生成（约 5-10 分钟）
5. 下载视频
```

### 方案 B：使用剪映 + AI 配音

#### 步骤 1：准备素材
```
需要准备：
- 产品界面截图/录屏
- 办公场景图片（可用 AI 生成）
- Logo 和品牌素材
- 背景音乐
```

#### 步骤 2：打开剪映
```
1. 下载安装剪映专业版（免费）
2. 新建项目
3. 设置分辨率：
   - 横屏：1920x1080
   - 竖屏：1080x1920
```

#### 步骤 3：添加素材
```
1. 导入所有素材
2. 按脚本顺序排列
3. 调整每个片段时长
```

#### 步骤 4：添加 AI 配音
```
1. 点击 "文本" → "识别字幕"
2. 或直接点击 "文本" → "文本朗读"
3. 粘贴脚本
4. 选择 AI 声音（推荐：商务男声/女声）
5. 生成配音
6. 调整与画面同步
```

#### 步骤 5：添加特效
```
1. 添加转场效果
2. 添加动画（入场、强调）
3. 添加字幕
4. 添加背景音乐（音量调至 20-30%）
```

#### 步骤 6：导出视频
```
1. 点击 "导出"
2. 选择分辨率：1080P
3. 选择格式：MP4
4. 等待导出完成
```

## 🚀 推广策略

### 第一阶段：内部推广（1-2周）

```
目标：让公司内部先用起来

行动：
1. 制作 1 个 3 分钟功能介绍视频
2. 在公司内部会议上播放
3. 发送到公司群和邮件列表
4. 邀请种子用户试用
5. 收集反馈和见证
```

### 第二阶段：社交媒体引流（2-4周）

```
目标：在社交平台建立声量

行动：
1. 每周发布 2-3 个短视频
   - 痛点共鸣视频
   - 功能展示视频
   - 使用技巧视频

2. 平台分布：
   - 抖音：主力平台，日活高
   - B站：技术人群多
   - 小红书：白领用户多
   - 视频号：企业决策者

3. 互动策略：
   - 评论区及时回复
   - 引导私信咨询
   - 定期直播答疑
```

### 第三阶段：口碑传播（持续）

```
目标：让用户帮你传播

行动：
1. 制作用户见证视频
2. 邀请 KOL/KOC 体验分享
3. 举办线上分享会
4. 建立用户社群
5. 用户推荐奖励机制
```

### 内容日历模板

```
周一：痛点视频（引发共鸣）
周三：功能视频（展示价值）
周五：用户见证/使用技巧
```
