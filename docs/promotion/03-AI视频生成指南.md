## 🎬 AI 视频生成步骤

### 方案 A：使用 HeyGen/Synthesia 生成数字人视频

#### 步骤 1：准备脚本
```
1. 使用上面的脚本模板
2. 或者用 ChatGPT 生成定制脚本：

提示词：
"请为我写一个 60 秒的产品介绍视频脚本。
产品是'智能知识库助手'，一个企业知识管理 AI 系统。
核心卖点：智能问答、越用越聪明、安全共享、AI 服务扩展。
目标受众：企业管理者和员工。
风格：专业但不枯燥，有感染力。"
```

#### 步骤 2：登录 HeyGen
```
1. 访问 https://www.heygen.com/
2. 注册/登录账号
3. 点击 "Create Video"
```

#### 步骤 3：选择数字人
```
1. 选择 "Avatar" 类型
2. 推荐选择：
   - 亚洲面孔（如 Angela、Michael）
   - 商务装扮
   - 自然表情
3. 或上传自己的照片生成专属数字人
```

#### 步骤 4：输入脚本
```
1. 将脚本粘贴到文本框
2. 选择语言（中文）
3. 选择声音（推荐专业、沉稳的声音）
4. 调整语速（建议 1.0-1.1x）
```

#### 步骤 5：添加素材
```
1. 添加背景（办公室场景）
2. 添加产品截图/录屏
3. 添加 Logo 水印
4. 添加字幕
```

#### 步骤 6：生成视频
```
1. 预览效果
2. 调整细节
3. 点击 "Generate"
4. 等待生成（约 5-10 分钟）
5. 下载视频
```

## ✅ 总结

### 最佳实践

1. **先有脚本，再做视频** — 脚本是灵魂
2. **开头 3 秒定生死** — Hook 要够吸引
3. **一个视频一个核心信息** — 不要贪多
4. **多平台适配** — 一个内容多种格式
5. **持续输出** — 量变引发质变
