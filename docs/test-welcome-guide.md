## 🔬 测试场景

### 测试 1: 首次访问自动显示

**步骤**:
```javascript
// 1. 打开浏览器控制台 (F12)
// 2. 清除引导完成标记
localStorage.removeItem('welcomeGuideCompleted');

// 3. 刷新页面
location.reload();
```

**预期结果**:
- ✅ 页面加载后自动显示引导遮罩层
- ✅ 显示渐变紫色的引导容器
- ✅ 显示"欢迎使用 RAG 2.0 知识演化系统"标题
- ✅ 显示5个进度点，第1个高亮
- ✅ 显示步骤 1 的内容（3个问题卡片）

**验证点**:
- [ ] 遮罩层背景模糊
- [ ] 容器有上滑动画
- [ ] 进度指示器正确
- [ ] 内容正确显示

### 测试 3: 跳过引导

**步骤**:
```
1. 点击右上角"跳过引导"按钮
2. 在确认对话框点击"确定"
```

**预期结果**:
- ✅ 显示确认对话框："跳过引导?"
- ✅ 点击确定后，引导页面关闭
- ✅ 显示主应用界面
- ✅ localStorage 设置完成标记

**验证点**:
```javascript
// 控制台验证
console.log(localStorage.getItem('welcomeGuideCompleted'));
// 应输出: "true"
```

**再次访问验证**:
```javascript
// 刷新页面
location.reload();

// 预期: 不再显示引导页面
```

### 测试 6: 中英文切换

**步骤**:
```javascript
// 1. 切换到英文
document.querySelector('button[onclick*="changeLanguage"]').click();

// 或者在控制台
window.LanguageModule.changeLanguage('en');

// 2. 重新打开引导
localStorage.removeItem('welcomeGuideCompleted');
location.reload();
```

**预期结果**:
- ✅ 标题变为："Welcome to RAG 2.0 Knowledge Evolution System"
- ✅ 按钮变为："Skip Guide", "Next", "Previous", "Get Started"
- ✅ 所有内容显示英文
- ✅ 步骤标题显示英文

**中文验证**:
```javascript
window.LanguageModule.changeLanguage('zh');
localStorage.removeItem('welcomeGuideCompleted');
location.reload();
```

**验证点**:
- [ ] 中文翻译正确
- [ ] 英文翻译正确
- [ ] 语言切换实时生效

### 测试 9: 性能测试

**测试指标**:
```javascript
// 1. 打开性能监控
performance.mark('guide-start');

// 2. 完整浏览一遍引导
// ...

// 3. 记录结束
performance.mark('guide-end');
performance.measure('guide-duration', 'guide-start', 'guide-end');

// 4. 查看结果
console.log(performance.getEntriesByName('guide-duration'));
```

**预期结果**:
- ✅ 首次加载 < 500ms
- ✅ 步骤切换 < 300ms
- ✅ 动画流畅 (60fps)
- ✅ 无卡顿

**验证点**:
- [ ] 加载速度
