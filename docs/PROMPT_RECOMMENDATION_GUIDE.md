# 高赞提示词推荐功能 - 快速使用指南

## 功能简介

当你在AI分析面板输入提示词时，不知道该怎么写？现在可以查看其他用户使用过的高评分提示词作为参考！

## 使用步骤

### 1. 打开AI分析面板
在文档列表页面勾选要分析的文档，点击右上角的 **"🤖 AI分析"** 按钮。

### 2. 选择分析策略
根据你的需求选择：
- 🚀 **快速了解大意** - 适合快速浏览
- 🔍 **精确查找答案** - 适合有具体问题
- 📊 **对比分析** - 适合比较多个文档
- 🎯 **深度分析** - 适合详细研究

### 3. 点击"高赞提示词"按钮
在提示词输入框右侧，找到橙色的 **"💡 高赞提示词"** 按钮并点击。

### 4. 浏览推荐提示词
弹出的窗口会显示该策略下的高评分提示词，每个提示词显示：
- ⭐ **评分**（1-5星）
- 🏷️ **策略标签**（带颜色区分）
- 🔥 **使用次数**
- 📝 **提示词内容**

### 5. 选择使用
直接 **点击任意提示词**，它会自动填充到输入框中。

### 6. 修改调整（可选）
根据你的具体需求，修改提示词内容。

### 7. 开始分析
点击 **"🚀 开始分析"** 按钮，执行AI分析。

## 示例场景

### 场景1：总结文档
**需求**: 快速了解一份技术文档的主要内容

1. 选择策略：**快速了解大意**
2. 点击 "💡 高赞提示词"
3. 看到推荐：
   - ⭐⭐⭐⭐⭐ "请详细总结这份文档的核心内容和关键观点。"
   - ⭐⭐⭐⭐⭐ "帮我快速提取文档的主要信息和重点。"
4. 点击使用第一个
5. 开始分析

### 场景2：查找特定信息
**需求**: 在文档中找到某个产品的价格

1. 选择策略：**精确查找答案**
2. 点击 "💡 高赞提示词"
3. 看到推荐：
   - ⭐⭐⭐⭐ "文档中提到的具体数字和数据是什么？"
   - ⭐⭐⭐⭐⭐ "请列出所有提到的价格信息。"
4. 点击第二个
5. 修改为："请告诉我产品A的价格。"
6. 开始分析

### 场景3：对比多个文档
**需求**: 对比三份竞品分析报告

1. 选择3份文档
2. 选择策略：**对比分析**
3. 点击 "💡 高赞提示词"
4. 看到推荐：
   - ⭐⭐⭐⭐⭐ "请对比这几份文档的异同点。"
   - ⭐⭐⭐⭐ "分析这些文档各自的优势和特点。"
5. 选择使用

## 推荐提示词的来源

高赞提示词来自：
- ✅ 历史问答中评分 **4星及以上** 的提示词
- ✅ 按 **评分高低** 排序
- ✅ 根据 **策略类型** 智能分类

## 策略类型说明

系统会自动检测提示词属于哪种策略：

| 策略类型 | 关键词示例 | 颜色标识 |
|---------|-----------|---------|
| 快速总结 | 总结、概括、summarize | 🔵 蓝色 |
| 深度分析 | 分析、详细、analyze | 🟠 橙色 |
| 对比分析 | 对比、比较、compare | 🟢 绿色 |
| 信息提取 | 提取、列出、extract | 🟣 紫色 |
| 精确查找 | 什么、哪个、what、which | 🔷 青色 |
| 通用 | 其他 | ⚪ 灰色 |

## 小技巧

### 💡 技巧1：学习优秀案例
浏览高赞提示词，学习如何更好地描述需求。

### 💡 技巧2：组合使用
选择一个推荐提示词，然后添加你的具体要求。

**示例**：
- 推荐："请详细总结文档的核心内容。"
- 修改为："请详细总结文档的核心内容，重点关注技术架构部分。"

### 💡 技巧3：记得评分
使用AI分析后，给出你的评分（😊/😐/😕），帮助改进推荐质量。

### 💡 技巧4：跨策略查看
如果当前策略推荐不多，可以切换到其他策略看看。

## 常见问题

### Q1: 为什么没有推荐提示词？
**A**: 可能的原因：
- 这是新系统，还没有足够的历史数据
- 当前策略下没有高评分记录
- 建议：切换到"全部策略"查看，或使用其他策略

### Q2: 推荐的提示词不适合我的需求怎么办？
**A**: 推荐提示词只是参考，你可以：
- 选择一个接近的，然后修改
- 直接输入自己的提示词
- 尝试其他策略的推荐

### Q3: 我的提示词会出现在推荐列表吗？
**A**: 会的！如果你的AI分析获得了 **4星或5星** 的评分，你使用的提示词就会成为其他用户的推荐。

### Q4: 如何提高我的提示词被推荐的机会？
**A**: 
- 写清晰、具体的提示词
- 分析后给予真实反馈
- 高质量的分析结果会获得高评分

## 最佳实践

1. **先选策略，再看推荐** - 推荐会根据策略自动筛选
2. **参考但不照搬** - 根据自己的文档调整提示词
3. **积极反馈** - 你的评分帮助改进推荐算法
4. **保持简洁** - 提示词不是越长越好
