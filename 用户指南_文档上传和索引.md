# 📖 文档上传和索引使用指南

## 🎯 快速开始

### 第一步：上传文档
1. 打开 AI 文档审查系统
2. 点击"🔍📁 文档搜索"标签页
3. 点击"选择文件"按钮，选择要上传的文档
4. 等待上传完成

### 第二步：建立索引
1. 上传完成后，注意查看**黄色提示框**
2. 点击"📊 统计信息"标签页
3. 如果显示警告："⚠️ 检测到未索引的文档"
4. 点击**"⚡ 立即执行增量索引"**按钮
5. 确认并等待索引完成

### 第三步：开始使用
1. 返回"💬 智能问答"标签页
2. 输入问题，系统现在可以搜索到新文档了！

## ❓ 常见问题

### Q1: 为什么上传文档后搜索不到？
**A:** 文档上传后需要建立索引才能被搜索。请前往"统计信息"页面执行"增量索引"。

### Q2: 什么是"增量索引"？
**A:** 增量索引只处理新增和修改的文档，速度快，推荐日常使用。

### Q3: 什么时候需要"重建索引"？
**A:** 以下情况建议使用重建索引：
- 首次使用系统
- 删除了大量文档
- 发现索引数据异常
- 迁移了文档存储位置

### Q4: "文档总数"和"已索引文档"有什么区别？
**A:** 
- **文档总数**：文件系统中的文档数量（包括新上传但未索引的）
- **已索引文档**：已经建立索引，可以被搜索的文档数量

### Q5: 索引需要多长时间？
**A:** 
- 增量索引：通常几秒到几分钟（取决于新文档数量）
- 重建索引：几分钟到十几分钟（取决于总文档数量）

### Q6: 可以同时上传多个文档吗？
**A:** 可以！选择文件时按住 Ctrl 键（Windows）或 Cmd 键（Mac）可以多选。

### Q7: 支持哪些文档格式？
**A:** 支持以下格式：
- 📊 Excel: .xlsx, .xls
- 📝 Word: .docx, .doc
- 📑 PowerPoint: .pptx, .ppt
- 📄 PDF: .pdf
- 📃 文本: .txt, .md
- 🌐 网页: .html, .xml

## 💡 最佳实践

### 1. 定期执行增量索引
- 建议：每次上传文档后立即执行
- 或者：每天定时执行一次

### 2. 文档命名建议
- 使用有意义的文件名
- 避免特殊字符
- 中英文均可

### 3. 文档组织建议
- 按主题分类文档
- 避免重复上传相同文档
- 定期清理无用文档

### 4. 性能优化
- 单次上传文件不超过20个
- 单个文件大小建议不超过50MB
- 批量上传后再执行索引，而不是每次上传都索引

## 🔍 统计信息解读

### 统计卡片说明

#### 📄 文档总数
- **含义**：文件系统中存储的所有文档数量
- **来源**：实时统计文档目录中的文件
- **用途**：了解系统中管理的文档数量

#### ✅ 已索引文档
- **含义**：已经建立搜索索引的文档数量
- **来源**：索引引擎统计
- **用途**：了解有多少文档可被搜索

#### ✅ 索引完成度
- **含义**：已索引文档占总文档的百分比
- **计算**：(已索引文档 / 文档总数) × 100%
- **理想状态**：100%

### 警告信息

当看到 **"⚠️ 检测到未索引的文档"** 警告时：
1. 说明有文档未建立索引
2. 这些文档无法被搜索到
3. 建议立即执行增量索引

## 🛠️ 故障排除

### 问题：上传失败
**可能原因**：
- 文件格式不支持
- 文件过大（超过限制）
- 网络连接问题

**解决方法**：
1. 检查文件格式是否在支持列表中
2. 尝试压缩大文件或分批上传
3. 检查网络连接

### 问题：索引失败
**可能原因**：
- 文档内容无法解析
- 系统资源不足
- 文件权限问题

**解决方法**：
1. 查看错误提示信息
2. 检查系统日志
3. 尝试删除问题文档后重新上传

### 问题：搜索结果不准确
**解决方法**：
1. 执行"重建索引"（完整重建）
2. 检查文档内容是否正确
3. 尝试使用不同的关键词
