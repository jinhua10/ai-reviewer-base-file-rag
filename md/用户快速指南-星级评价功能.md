# 用户快速指南 - 星级评价功能

> **5 秒学会使用** | 2025-11-30

---

## 🌟 新功能：文档质量评价

### 在哪里找到？

当您提问后，在**参考来源**区域，每个文档都有三个按钮：

```
┌─────────────────────────────────────┐
│ 1. 培训通知.pdf         [下载] │
│ ────────────────────────────────│
│ [👍 有帮助]  [👎 无帮助]  [⭐评价质量] │
└─────────────────────────────────────┘
```

---

## 💡 如何使用？

### 方式1：快速反馈（原有功能）

- **点击 "👍 有帮助"**：文档对您有帮助
- **点击 "👎 无帮助"**：文档没什么用

**效果**：小幅调整推荐权重

---

### 方式2：星级评价（新功能）🎉

**点击 "⭐ 评价质量"** 打开评价窗口：

```
╔═══════════════════════════════════════╗
║     💎 评价文档质量                    ║
║                                       ║
║  这个文档对回答问题有多大帮助？          ║
║                                       ║
║      ★  ★  ★  ★  ★               ║
║                                       ║
║         😐 一般般                     ║
║                                       ║
║  [可选：说说您的想法...]                ║
║                                       ║
║     [取消]        [提交评价]           ║
╚═══════════════════════════════════════╝
```

### 点击星星选择评分：

| 星级 | 含义 | 系统反应 |
|------|------|----------|
| ⭐ | 😞 完全没用 | 大幅降低推荐 ⚠️ |
| ⭐⭐ | 🙁 帮助不大 | 降低推荐 📉 |
| ⭐⭐⭐ | 😐 一般般 | 保持不变 ➡️ |
| ⭐⭐⭐⭐ | 😊 很有用 | 增加推荐 📈 |
| ⭐⭐⭐⭐⭐ | 🤩 非常有用 | 优先推荐 🚀 |

**效果**：大幅调整推荐权重（5倍于快速反馈）

## ❓ 常见问题

**Q: 评价后还能修改吗？**
A: 目前每个文档只能评价一次。请谨慎选择星级。

**Q: 如果不小心点错了怎么办？**
A: 点击"取消"按钮即可退出，不会提交评价。

**Q: 必须写评论吗？**
A: 不必须。评论是可选的，但会帮助我们更好地理解您的想法。

**Q: 评价会立即生效吗？**
A: 是的！提交后立即影响文档推荐优先级。

**Q: 我能看到别人的评价吗？**
A: 目前不显示单个评价，但会看到平均评分和评价人数。
